# ScanPeek Privacy Policy

This repository hosts the privacy policy page for **ScanPeek**, a QR code scanner app.

The policy is published as a static HTML page and is available at the GitHub Pages URL linked to this repository.

## Overview

ScanPeek does not collect or store any personally identifiable information. All QR code scanning is processed locally on the device.

### Data handled

| Data | Where it stays |
|------|----------------|
| Camera input | On-device only, never stored or transmitted |
| Scanned content | On-device only, never sent to servers |
| Gallery images | On-device only, never stored or transmitted |
| Safe browsing credits | Local storage (SharedPreferences) |

### Third-party services

- **Google Safe Browsing API** — scanned URLs are forwarded to check for threats
- **Google AdMob** — used to display advertisements; may collect device identifiers

### Permissions

- **Camera** — required to scan QR codes
- **Storage / Media access** — required to load QR code images from the gallery
- **Internet** — required for Safe Browsing checks and loading ads
